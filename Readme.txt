# 📚 Smart Library Analytics Dashboard

This project presents an interactive **Power BI dashboard** designed to analyze library usage patterns, book demand, and borrowing trends.

## 🚀 Features

* 📊 Library performance overview with KPI metrics
* 📈 Monthly borrowing trend analysis
* 🔮 Borrowing demand forecasting using Power BI forecasting
* 📚 Book popularity analysis (Top & Least borrowed books)
* 🔍 Interactive **Book Explorer** to search and analyze individual books
* 🧠 Department-level borrowing insights

## 📊 Dashboard Pages

### 1️⃣ Library Overview

Shows overall library statistics including total books, members, borrowings, fines, and genre popularity.

### 2️⃣ Book Analytics & Demand Prediction

Analyzes popular books, author demand, and predicts future borrowing trends.

### 3️⃣ Book Explorer

Allows users to search for a specific book and view detailed borrowing insights and department usage.

## 🛠 Tools Used

* Power BI
* Excel Dataset
* Data Visualization
* Predictive Forecasting
